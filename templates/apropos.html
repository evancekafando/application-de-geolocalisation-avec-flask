A propos

Geolocalisation-app est une application Web basée sur la localisation créée en utilisant le framework Web Flask. L'application utilise les principales fonctionnalités de Flask, telles que les modèles, les vues, les contrôleurs, les modèles Web, les formulaires, les validations et même l'interaction avec une API.
